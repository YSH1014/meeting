### 部署
* 软件环境准备：Python3,pip  
* 安装virtualenv: ```python3 -m pip install virtualenv```
* 建立virtualenv:```python3 -m virtualenv virtualenv```
* 使用virtualenv的python环境：```. virtualenv/bin/activate```
* 安装运行必备包```pip install -r requirements.txt```
